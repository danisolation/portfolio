Forsaken

Introduction

This project, developed by students of the Ho Chi Minh City University of Technology, aims to create a smart home monitoring and control system. This system leverages advanced technologies to provide users with convenient, efficient, and secure control over their home environment.

The rapid development of information technology and artificial intelligence enables the creation of a comfortable, safe, and secure living environment by maximizing the use of smart devices and sensors.

Features

  • Device Monitoring: Monitor the status of various devices in the house, including lights, air conditioners, fans, security cameras, automatic doors, temperature sensors, humidity sensors, motion sensors, and fire detectors.
  • Device Control: Control home devices, allowing users to turn lights on/off, adjust temperatures, open doors, turn on TVs, close curtains, etc.
  • Automation: Automate devices by setting schedules and rules for device operation, helping users save time and energy.
  • Security Alerts: Provide security alerts and protection, allowing users to detect and notify security incidents such as intrusions, fires, or gas leaks.

Setup Instructions

To get started with Forsaken, follow these steps:

Server Setup

  1. Ensure Node.js is installed on your machine.
  2. Install the necessary node_modules by running npm install.
  3. Add the AIO Key to the .env file:
  4. Start the server by running node app.js. The server will run on PORT 8080.
