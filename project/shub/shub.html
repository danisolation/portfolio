Shub

Introduction

Shub is a document storage and management system built on the Blockchain platform, providing a secure and transparent solution for managing your organization's confidential documents.

The need for storage has emerged and existed in human society since the early days of mankind, when civilization was just beginning to emerge. In today's modern society, the need for storage has become an essential issue for individuals and even more important for organizations. Each organization has important documents that need to be kept confidential. However, the platforms and systems that organizations often use today still rely on third parties, leading to a lack of transparency, difficulty in control, and the risk of data leakage.

Features

  • User Management: Create, manage, and assign permissions to user accounts within the organization.
  • Document Management: Securely store, organize, and manage digital documents, ensuring confidentiality and integrity.
  • Folder Management: Create and organize folders to categorize and structure documents efficiently.
  • Sharing and Collaboration: Share documents securely with other users within the organization, facilitating collaboration and teamwork.

Setup Instructions

To get started with Shub, follow these steps:

Prerequisites

  • https://hyperledger-fabric.readthedocs.io/en/release-2.5/prereqs.html

Run network

  • cd test-network
  • ./network.sh down
  • ./network.sh up createChannel -c mychannel -ca -s couchdb
  • ./network.sh deployCC -ccn basic -ccp ../shub/back-end/chaincode/ -ccl typescript

Run IPFS

  • cd ipfs-network
  • ./network.sh down
  • ./network.sh up

Run API

  • cd shub/back-end/api
  • npm install
  • npm run start or npm run start:dev (for hot reload)

Run Front-end

  • cd shub/front-end
  • npm install
  • npm run dev
